 # 时间戳转换函数封装





* ### [formatDate](https://github.com/CloudEmperor/blog/blob/master/utils/formatDate/formatDate.js)
>
> 时间戳转换函数封装，方便使用。
> 
>
>
>
> 使用方法：
>
> ```javascript
>
>  //引入
>
>  import { formatDate } from "@/utils/formatDate";
>
>  //转换
>  formatDate(value, "yyyy-MM-dd  hh: mm : ss")
>                           
>
>```
>
